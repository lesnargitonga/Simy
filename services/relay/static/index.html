    Advanced Tools

    Less common relay actions and presentation helpers.

    Add Contact

    Import a share artifact or enter details manually. Mailbox cards identify a workspace, pairing links establish a fresh secure relationship, and direct contact links carry an already chosen shared secret.

    Mailbox cards prefill identity only. Pairing links can complete secure contact setup immediately.

    Paste an existing relay mailbox ID, or create a real relay mailbox for a new contact.

    Use the same secret on both sides when creating a direct contact link for this relationship.

    Contact Link

    Send this direct contact link out of band. It includes your mailbox ID and a specific shared secret for this relationship.

    Share Workspace

    Copy your mailbox card or generate a fresh one-to-one pairing link. Pairing links do not create a new mailbox.

    This shares your existing mailbox identity only. It does not include a shared secret.

    Use this when the other person already has a workspace. It creates a fresh shared secret for one new relationship and they will send back a reply link.

    Send this reply link back to finish the two-way pairing.

    Provisioned User Pack

    This pack already includes the UUID mailbox and retrieval token needed to onboard a new account. The suggested codename is only a starting point and can still be changed by the user during setup.

    Admin Access

    Use the relay admin token from the workspace .env file under RELAY_ADMIN_TOKEN.

    This is the live relay admin secret from the root .env file, not the mailbox token shown on the setup screen.

    Repair Secure Link

    Paste an invite link, a provision pack, a saved credential pack, or the raw shared secret for this contact.

    The app will attach the secret only to this local contact on this device.
